##摘要
    本文主要对物理摆的混沌现象进行一些探讨与研究。
##背景介绍
    混沌现象通常是指发生在确定性系统的随机的不规则运动，就是说在一个确定性理论描述的系统中，其表现出来的行为却是一种不可重复、不可预测的带有不确定性的这一现象。
    进一步研究表明：混沌是非线性动力系统普遍存在的现象。
